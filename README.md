# Claude-Signal-Lamp
Win11 ClaudeCode Cli  Desktop floating traffic light that shows Claude Code's working status in real-time.    中文版：    桌面悬浮红绿灯，实时显示 Claude Code 工作状态    README 开头介绍（如果你想加一段）：    Claude Code 没有提供可视化的运行状态指示。这个小工具在桌面右下角放一个悬浮红绿灯，通过 Claude Code 的 hook   系统实时反映当前状态：绿灯空闲、黄灯工作中、红灯等待确认。不用盯着终端，余光就能知道它有没有在干活。
